# RaidCopy

WoW 1.12.1 addon. Grabs your raid roster and puts it in a copyable text window.

## Usage

Click the minimap button to open. Text is pre-selected — Ctrl+C and paste into spreadsheet.

## Output format

`Name,CLASS; Name,CLASS; ...`

Sorted alphabetically, up to 40 players.

## Commands

`/raidcopy` or `/rc` — toggle the window
